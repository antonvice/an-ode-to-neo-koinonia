# Interfaces

The chat interface was not the final interface to artificial intelligence. It was the first discovered handle.

Language let ordinary people touch the phenomenon. Agents let the phenomenon touch tools. The next step is not longer chat, but tighter coupling between intention, state, tools, and generated interface.

The computer stops being a set of apps and becomes a loop: intention in, agent reasoning, tool action, state update, interface out.

In that loop, interface is no longer a fixed surface. It is a temporary shape produced around the task, the available data, the permission boundary, and the user's current intention.
