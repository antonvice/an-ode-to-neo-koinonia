# Operating System

The operating system was built for direct manipulation: windows, files, applications, devices, permissions, processes.

An agentic computer changes the center of gravity. The user no longer needs to personally traverse every tool. The system can interpret intention, call capabilities, retrieve context, update state, and render the next useful interface.

The old OS does not disappear immediately. It becomes the training ground and scaffolding for the next one.
