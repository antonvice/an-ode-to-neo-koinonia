# Model Inference

In the post-OS computer, model inference becomes the active runtime.

The old computer runs applications. The agentic computer runs transformations between intention, state, permission, tool action, and generated UI.

This does not mean everything becomes a single model. It means inference becomes the coordinating act: the place where the system decides what state matters, what tool should be called, what must be confirmed, and what interface should appear next.
