# Voice Interface

Voice is a transitional interface between typing and intention.

Dictation makes visible how little manual interaction remains necessary once AI can absorb language, hold context, and operate tools. The user does less keyboard work and more intention transmission. The computer becomes less like a surface to manipulate and more like a listener that can act.

Voice is still language, so it still carries the limits of language. But it loosens the body from the keyboard and reveals the direction of travel: less clicking, less typing, more direct communication of intent.

The next step is not merely better speech recognition. It is an interface that understands the current state, the focused task, the available actions, and the user's desired transformation.
