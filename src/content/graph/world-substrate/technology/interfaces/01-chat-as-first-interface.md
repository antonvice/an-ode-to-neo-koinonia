# Chat as the First Interface

Chat was the first interface through which ordinary people touched artificial intelligence.

This was natural. Language is the human universal API. It carries requests, memory, emotion, ambiguity, commands, and explanation. When AI first became useful, the easiest way to expose it was to put a text box in front of it.

But chat is not the final form.

Chat is slow, sequential, and trapped inside language. It makes the user translate intention into words, then makes the model translate action back into words. This works for explanation, but it is clumsy for operation. A person does not want a paragraph about a flight search when they need a living set of filters, maps, choices, prices, and confirmations.

Chat revealed the phenomenon. Agents extended it. The next interface must stop treating AI as a talking document and start treating it as an operating intelligence.
